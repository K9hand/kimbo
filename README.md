# Kimbo IT Pty Ltd Website

Simple one-page marketing website for Kimbo IT Pty Ltd, built with vanilla HTML, CSS, and a small amount of JavaScript for the mobile navigation.

## Files

- `index.html` - semantic page structure and content
- `style.css` - responsive layout, design system, and component styles
- `script.js` - mobile menu toggle behavior

## Run

Open `index.html` directly in any modern browser.

## Customize

- Replace the placeholder phone, email, and physical address in `index.html`
- Point the form to a real backend or form handler when ready
- Update metadata in the `<head>` if final business details change

## Notes

- The site uses section anchors for one-page navigation
- Smooth scrolling is enabled and reduced-motion preferences are respected
- No external frameworks or build tools are required
